# HBTI 16型人格测试 — App Store 上架准备清单

## 一、开发者账号准备

- [ ] 使用公司域名邮箱注册 Apple ID
- [ ] 开启 Apple ID 双重认证
- [ ] 注册 Apple Developer Program（标准计划，688元/年）
- [ ] 使用 D-U-N-S 编码完成公司身份验证
- [ ] 等待苹果审核通过（3-5个工作日）

## 二、Xcode 项目配置

- [ ] Bundle Identifier: `com.huichengjia.hbti`
- [ ] 最低支持版本: iOS 17
- [ ] 添加 App Icon 到 Assets.xcassets（1024x1024 主图标）
- [ ] 配置 Launch Screen
- [ ] 在 Signing & Capabilities 中配置 Team
- [ ] 在 Info.plist 中添加隐私权限说明（如有需要）

## 三、App Store Connect 配置

- [ ] 创建新 App，填写 App 名称：HBTI 16型人格测试
- [ ] 副标题：探索你的性格类型，发现真实的自己
- [ ] 主要类别：生活方式
- [ ] 次要类别：健康健美
- [ ] 年龄分级：4+
- [ ] 配置 IAP 产品：
  - `hbti_report_48` — 48题详细报告 — 9.9元
  - `hbti_report_93` — 93题详细报告 — 29.9元
- [ ] 填写隐私政策 URL
- [ ] 在隐私问卷中声明收集的数据类型

## 四、隐私政策页面（部署到阿里云）

- [ ] 在阿里云服务器上部署隐私政策页面
- [ ] URL 格式：`https://yourdomain.com/privacy`
- [ ] 内容包含：收集信息、使用方式、存储方式、用户权利、联系方式

## 五、App 截图准备

需要准备以下尺寸截图（每套 5-10 张）：

- [ ] iPhone 6.7" 显示屏（iPhone 15 Pro Max / 14 Pro Max）— 1290 x 2796
- [ ] iPhone 6.1" 显示屏（iPhone 15 / 14）— 1179 x 2556

截图内容建议：
1. 欢迎页（用户名输入 + 版本选择）
2. 答题界面（滑动量表 + 情境插图）
3. 基础结果页（类型代码 + 四维解析）
4. 详细报告页（职业推荐 + 发展建议）
5. 分享卡片预览
6. 历史记录页
